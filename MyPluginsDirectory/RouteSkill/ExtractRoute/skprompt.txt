# タスクについての記述
1. <input>タグに挟まれた入力の文章の中から、出発地と到着地の地名を抽出してください。
2. departure に出発地、destination に到着地を記載して、以下の例の様に出力してください。
3. 駅名の判別がつかなかった場合には、それぞれの駅名の出力場所に unknown と出力してください。

# 例1
<input>
品川駅から新橋駅まで行きたいです。
</input>

Response:
{
    "departure" : "品川駅",
    "destination" : "新橋駅"
}
